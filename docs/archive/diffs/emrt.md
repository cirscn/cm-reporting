# EMRT 模板（1.1–2.1）差异与逻辑说明（产品可用版）

> 基线版本：EMRT 2.1（本地模板解析）

**申报范围 → 产品清单 联动**
- 申报范围=Product (or List of Products)：产品清单 必填（至少 回复方的产品编号）
- 申报范围=User defined：必须填写范围描述

**通用填写约束（来自 Instructions）**
- 填写表格时，单元格条目不应以“=”或“#”开头（建议提示，可选校验）。

**DV 文案（非日期）**
- 申报范围下拉错误提示：errorTitle=“Required Field”；error=“Select from dropdown options to declare survey scope”。  

**矿种选择联动（关键）**
- 选择矿种后，Questions 1–7 与 C 会按字母顺序生成对应矿种行。
- Delete [Mineral] 会移除该矿种行；答案位置不自动重排，需要人工校对。
- 模板提示：问题 1–7 与 C 自动填充所选矿产并按字母顺序列出；请检查回答是否与对应矿产一致。

**说明（来自 Instructions，产品层面仅提示）**
- 公司信息 / 申报问题 / A–G / Mine List（2.0+）模板文字要求仅以英文作答；中文环境验证仍以英文选项值为准（产品仅提示不强制）。
- 模板要求填写完毕日期为 DD-MMM-YYYY；文件名示例为 companyname-date.xlsx（date as YYYY-MM-DD，示例格式与日期字段不同）。
- 填写“冶炼厂联系人姓名/电子邮件”前，需确认已取得联系人同意（模板提示）。
**术语提示（产品释义）**
- CAHRA：受冲突影响和高风险地区。
- IPC1755：供应链尽职调查信息交换标准（IPC 1755）。

## 2. 申报问题（1–7）—矿种矩阵
> 结构：每道题都要对 **申报范围内矿种** 分别回答；矿种行由 Declaration 的“矿产申报范围”自动填充并按字母顺序排列。

**适用矿种**
- 仅显示已在 Declaration 中选择的矿种（当前模板候选：Cobalt / Copper / Graphite / Lithium / Mica / Nickel）。

**Q1 是否在产品或生产流程中有意添加或使用任何指定矿产？（必填）**
- 每个矿种一行：1.1–1.3 回答 = Yes / No / Unknown / Not applicable for this declaration；2.0+ 回答 = Yes / No / Unknown / Not declaring；注释 可填

**Q2 产品中是否还存有任何指定矿产？（必填）**
- 每个矿种一行：回答 = Yes / No / Unknown；注释 可填

**Q3 是否从 CAHRA 采购指定矿产？（必填）**
- **1.x**：Cobalt 回答 = Yes / No / Unknown / DRC only；Mica 回答 = Yes / No / Unknown / India and/or Madagascar only；注释 可填
- **2.0+**：每个矿种一行：回答 = Yes / No / Unknown；注释 可填

**Q4 是否 100% 来自回收/报废料？（必填）**
- 每个矿种一行：回答 = Yes / No / Unknown；注释 可填

**Q5 相关供应商响应率是多少？（必填）**
- 每个矿种一行：**1.x** 回答 = 100% / Greater than 90% / Greater than 75% / Greater than 50% / 50% or less / None（模板中 100% 对应值为 1）；**2.0+** 回答 = 100% / Greater than 90% / Greater than 75% / Greater than 50% / 50% or less / None / Did not survey；注释 可填

**Q6 是否识别出全部冶炼厂或加工厂？（必填）**
- 每个矿种一行：回答 = Yes / No / Unknown；注释 可填

**Q7 是否已在声明中报告全部冶炼厂/加工厂？（必填）**
- 每个矿种一行：回答 = Yes / No / Unknown；注释 可填

**依赖关系（来自 Instructions）**
- Q3–Q7：仅当对应矿种 Q1=Yes 且 Q2=Yes 时才需要回答。
- 若任一矿种在 Q1/Q2 为 Yes，则必须回答公司层面问题 A–G。
- 模板提示：问题区回答模板文字要求仅以英文作答；产品层面仅提示（不强制校验）。

> 结构：每题一个 回答（下拉）+ 注释（备注）

| 题号 | 题干（模板原文） | 回答 允许值 |

| B | 贵公司的负责任矿产采购政策是否公开发布于贵公司网页上？（备注 - 如果是，请在注释字段中注明 URL。）(*) | Yes / No |
| C | 贵公司是否要求贵公司的直接供应商从尽职调查实践经独立第三方审计计划验证过的冶炼厂采购指定矿产？(*) | Yes / Yes, when more processors are validated / No |
| D | 贵公司是否已实施负责任矿产采购的尽职调查措施？(*) | Yes / No |
| E | 贵公司是否针对相关供应商进行指定矿产供应链调查？(*) | Yes, in conformance with IPC1755 (e.g. EMRT) / Yes, Using Other Format (Describe) / No |
| F | 贵公司是否会根据预期审核从供应商处收到的尽职调查信息？(*) | Yes / No |
| G | 贵公司的审核流程是否包括纠错行动管理？(*) | Yes / No |

**条件提示（来自条件格式）**
- B 题若回答 Yes → 注释需提供 URL。
- E 题若回答 “Yes, Using Other Format (Describe)” → 注释必填。

> 版本差异说明：  
> - **1.1**：A 为“钴采购政策”；B/D 为“政策是否覆盖 OECD 附录二风险/童工问题”等表述；C/E 仍指“钴/云母”。  
> - **1.11–1.3**：A/B/D 更新为“矿产采购政策/官网公开/已实施尽调”；**C/E 仍指钴/云母**。  
> - **2.0+**：C/E 改为“指定矿产”，其余 A/B/D/F/G 维持矿产表述。  

## 4. Smelter List（冶炼厂清单）
**关键逻辑（来自 Instructions）**
- 冶炼厂查找为下拉字段，包含已知冶炼厂，以及 **Smelter not listed / Smelter not yet identified** 两个特殊选项。
  - **Smelter not listed**：需手工填写 冶炼厂名称，并手选 冶炼工厂地址（国家）（模板明确说明）。
- **Smelter not yet identified**：在冶炼厂查找清单中该项的标准冶炼厂名称=Unknown、国家为空；不会自动回填到冶炼厂名称/冶炼工厂地址（国家）；**2.1 起**触发国家列提示，2.0/1.x 无提示。
  - 备注：L 页多语言说明称 D/E 自动填 Unknown，但与 Look‑up 数据/公式不一致；以 Look‑up + 条件格式为准（国家为空并触发提示）。
- 若已知冶炼厂识别号码，可先填 冶炼厂识别号码输入列，其他信息自动填充。
- 为触发查找，需先填写 金属 → 冶炼厂查找（从左到右）。
- 金属下拉来源于 Declaration 的申报矿种动态列表（按已选矿种生成），只包含本次申报范围内矿种；未纳入矿种无法在本表选择。
- 若贵公司为冶炼厂/精炼厂，模板建议在冶炼厂清单中填写本公司信息（产品层面提示）。

**填写路径（来自 Smelter List 说明）**
- 选项 A：已知冶炼厂识别号码 → 在“冶炼厂识别号码输入列”输入；其余列自动填充，“冶炼厂名称”变为灰色。
- 选项 B：已知金属 + 冶炼厂查找名称 → 先选金属，再从“冶炼厂查找”下拉选择。
- 选项 C：已知金属 + 冶炼厂名称 → 在“冶炼厂查找”选择“冶炼厂未列出”，再手工填写“冶炼厂名称”“冶炼工厂地址（国家）”，并补齐其他信息列。

**与申报区联动（模板说明/校验）**
- 金属下拉受 Declaration 申报矿种列表控制（动态列表）；未纳入矿种不会出现在下拉中。
- 金属已选但“冶炼厂查找”为空时会触发高亮提示（必填校验）。
- “冶炼厂查找”下拉会随所选金属过滤。
- **1.3**：若金属为 Cobalt/Mica 且 Q1/Q2 对应矿种回答为 No/Unknown/Not applicable for this declaration → 金属/查找/名称/国家列触发提示（条件格式）。
- 选择“Smelter not listed”时，冶炼厂名称与冶炼工厂地址（国家）为空会触发高亮提示（红色提示）。
- 2.1：选择“Smelter not yet identified”时，国家列会触发提示（红色提示）；冶炼工厂地址（国家）不会自动回填，需手工补充。  
- 2.0/1.x：不触发国家列提示（国家为空，且不自动回填）。

## 7. Checker 校验规则（必填与联动）
**Checker 明确列为必填（F=1）**  
- 公司信息：公司名称、申报范围或种类、矿产申报范围、联系人/授权人信息、授权日期。  
- Q1–Q7：申报范围内矿种逐题回答（每题每矿种）。  
- 公司层面问题 A、B、D–G：必填回答。  
- 公司层面问题 C：按矿种逐一回答（矿种矩阵）。  
- Smelter List：对每个申报矿种提供冶炼厂/加工厂清单。  

**条件必填（来自 Instructions/Declaration/Smelter 规则）**  
- 申报范围=Product (or List of Products) → Product List 必填（至少“回复方的产品编号”）。  
- 申报范围=User defined → 范围描述必填。  
- Q3–Q7：仅当对应矿种 Q1=Yes 且 Q2=Yes 时要求填写（模板说明）。  
- B 题若回答 Yes → 注释需提供 URL（模板提示）。  
- Smelter List 中选择 “Smelter not listed” 的行 → 冶炼厂名称与国家/地区必填。
