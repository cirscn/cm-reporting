# Definitions 精选术语摘要（产品可读版）

> 目的：为产品经理提供“可读的核心术语速览”。  
> 全量定义见：`docs/prd/appendix-definitions.md`；术语映射主表见：`docs/prd/definitions-master.md`。

## 1. 核心术语（跨模板通用）
| 术语 | 简释（产品口径） | 建议出现位置 |
|---|---|---|
| 冲突矿产（Conflict Minerals） | 法规定义的冲突矿物范围；CMRT 主要涉及 3TG。 | 模板简介 / 申报说明 |
| 3TG | 钽(Tantalum)/锡(Tin)/钨(Tungsten)/金(Gold)。 | CMRT 问题区 |
| CAHRA | 受冲突影响和高风险地区。 | Q3/Q4/公司层面问题 |
| SEC | 美国证券交易委员会；Dodd‑Frank 1502 监管主体。 | CMRT 问题区/说明 |
| Dodd‑Frank 1502 | 冲突矿产披露法规来源。 | CMRT Definitions |
| OECD 尽职调查 | 负责任供应链尽职调查指南。 | 公司层面问题/Definitions |
| RMI / RMAP | 责任矿物计划与其审验/评估标准。 | Smelter List / Definitions |
| 冶炼厂 / 精炼厂（Smelter/Refiner） | 供应链关键节点；Smelter List 需填报。 | Smelter List |
| 回收/报废料（Recycled/Scrap） | 来自回收材料或报废料的来源描述。 | Smelter/Mine List |
| IPC1755 | 冲突矿产数据交换标准；在公司层面问题中引用。 | 公司层面问题 |

## 2. 模板特有术语（摘要）
> 以下为模板内“金属/加工厂”定义的代表性类别，完整文本请查附录。

- **EMRT**：钴精炼厂、石墨生产商、天然石墨、加工商等。  
- **AMRT 1.1–1.3**：铝/铜/锂/镍/锌/稀土等加工厂定义。  
- **CRT**：钴供应链专用术语（钴冶炼厂/尽职调查口径）。

## 3. 产品展示建议
- **术语提示优先级**：问卷题干 > 申报范围 > 清单字段。  
- **强提示**：CAHRA / SEC / OECD / RMI/RMAP / IPC1755（影响问题理解）。  
- **弱提示**：金属加工厂定义（仅在用户停留时展示 Tooltip）。
