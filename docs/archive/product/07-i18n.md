# 国际化与语言口径

## 语言原则
- 模板 Instructions 通常要求**英文作答**；中文界面仅做展示与提示。
- 校验口径以英文原值为准（避免翻译导致的规则失配）。

## 资源来源
- 中文：`docs/i18n/zh-CN.json`
- 英文：`docs/i18n/en.json`

## 处理建议
- 选项值：保存英文原值；中文仅用于展示层。
- 字段名：显示中文 + 保留英文原字段名（便于对齐模板与导出）。
- 术语与缩写：以 `docs/prd/definitions-summary.md` 与 `docs/prd/definitions-master.md` 为准。
