# PR 模板（工程实现期）

## 变更概述
- [ ] 变更目的清晰且可追溯到 PRD/规则证据
- [ ] 影响范围已说明（模板/版本/模块）

## 架构一致性
- [ ] registry/规则/transform 分层无交叉污染
- [ ] ui 无模板/版本分支判断
- [ ] i18n key 完整且 en/zh 同步

## 性能检查（Vercel）
- [ ] 无串行 await 可并行的路径
- [ ] 无 barrel import、重依赖按需加载
- [ ] 列表/表格支持虚拟化或 content-visibility

## 测试与验证
- [ ] 核心规则/校验已覆盖（单测或手工核对路径）
- [ ] 关键页面可运行与可定位错误

## 风险与回滚
- [ ] 风险点说明
- [ ] 回滚路径说明
